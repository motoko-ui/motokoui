# Motoko UI

Everything you need to build modern React applications.

## Documentation

Visit `/docs` locally after starting the app.

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

```
apps/www          # Next.js site (landing + docs)
skills/           # Agent skills for Motoko UI
.github/          # CI, issue & PR templates
packages/         # Shared packages (placeholder)
```
